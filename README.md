## What is Keystor.es?
Keystor.es is a basic RoR service that enables customers to safely and securely store their keystore files (typically Android dev) in the cloud.

## Installation

Ruby 2.3.3

Rails 5.0.0

PostgreSQL 
 
Keystor.es also uses AWS S3
